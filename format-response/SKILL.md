# CLI 응답 포맷 가이드

줄줄이 텍스트는 안티패턴이다. 섹션을 나누고 패턴을 골라 쓴다.

## Claude Code CLI 렌더링 특성

| 요소 | 렌더링 여부 |
|---|---|
| `**bold**`, `# 헤더`, `- 목록` | ✅ 마크다운 렌더링됨 |
| 코드블록 (` ``` `) | ✅ monospace, 정렬 유지 |
| 마크다운 표 `\| col \| col \|` | ❌ 파이프 문자 그대로 출력 |
| 유니코드 박스 문자 (코드블록 안) | ✅ 그대로 출력, 정렬 유지 |

**결론:** 표는 마크다운 표가 아니라 유니코드 박스를 코드블록 안에 넣는다.

---

## 패턴 선택 기준

- **도구·프로젝트 시작, 섹션 강조** → 배너 (이중선 박스)
- **빌드·테스트 결과, 상태 요약** → 배지 `[OK]` `[WARN]` `[ERROR]`
- **섹션 구분** → 디바이더 `──`
- **비교 (선택지, 옵션)** → 유니코드 박스 테이블 (코드블록 안)
- **파일·디렉터리 구조** → 트리 `├── └──`
- **사람 결정 필요 항목** → 격리 박스
- **진행률** → 프로그레스 바 `[████░░]`
- **완료 목록 + 진행 중 혼합** → 완료 위 고정 / 진행 아래 동적
- **다음 액션 선택지** → 하단 커맨드 바

---

## 원칙

- **단락 3줄 초과 = 구조화 대상** — 줄줄이 텍스트 금지
- **섹션마다 디바이더** — 눈이 자동으로 끊어 읽는다
- **결정 사항은 격리** — 사람이 판단해야 할 항목은 박스로 분리
- **상태는 배지로** — `[OK]` `[WARN]` `[ERROR]` 한 줄 처리
- **표는 코드블록 안 유니코드 박스** — 마크다운 표는 CLI에서 렌더링 안 됨
- **Python/Rich 스크립트 금지** — 코드 생성 아닌 텍스트로 직접 출력

## 스니펫

**배너**
```
╔══════════════════════════════════════╗
║         프로젝트 / 도구 이름         ║
║         부제목 또는 버전 정보        ║
╚══════════════════════════════════════╝
```

**배지**
```
[OK] 빌드 성공 125 페이지   [WARN] 경고 2건   [ERROR] 오류 1건
```

**디바이더**
```
── 섹션 제목 ───────────────────────────────────
```

**유니코드 박스 테이블**
```
┌──────────────┬────────┬────────┐
│ 할 일        │ 시간   │ 긴급도 │
├──────────────┼────────┼────────┤
│ 집 청소하기  │ 2시간  │ 🔴 높음 │
│ 책 정리하기  │ 30분   │ 🟡 보통 │
│ 침구류 정리  │ 10분   │ 🟢 낮음 │
└──────────────┴────────┴────────┘
```

**격리 박스 (사람 결정 필요)**
```
┌──────────────────────────────────────────────┐
│  [1] 히어로 헤드라인 — PRD §6.1 vs 현재 잠정 │
│  [2] 브랜드 표기 — 정처기 vs jeongcheoki     │
└──────────────────────────────────────────────┘
```

**프로그레스 바**
```
빌드 중...  [████████████░░░░░░░░] 63%
```

**완료 + 진행 혼합**
```
[완료] 항목 A 처리됨
[완료] 항목 B 처리됨
──────────────────────────────────
처리 중...  [████████░░░░] 67%
```

**트리**
```
project/
├── src/
│   └── main.py
└── README.md
```

**하단 커맨드 바**
```
══════════════════════════════════════════════
  [A] 카피 개정   [B] 인터랙티브   [C] 커밋
══════════════════════════════════════════════
```
